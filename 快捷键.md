- Xcode 快捷键

  旋转模拟器 cmd+左右
  前进后退代码 ctrl+cmd+左右
  切换.h.m ctrl+cmd+上下
  查找 shift+cmd+F

  shift+command+o 快速查找类
  shift+command+J 定位当前编辑的文件

  

- AppCode快捷键
  参数提示 opt+cmd+P 
  快速修复 opt+enter
  查找类 cmd+O

  

- AS快捷键
  ---------   Win    --------
  竖选切换 alt+shift+insert

  选中下一个相同内容 alt+J

  --------    IOS    -------

- Mac快捷键

  ^ == Ctrl
  Z- == option
  up == shift
  left == tab
